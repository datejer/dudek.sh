## develop.

```bash
$ git clone https://github.com/datejer/dudek.sh.git
$ cd dudek.sh
$ pnpm install
$ pnpm dev
```

## lint.

Using ESLint and Prettier.

```bash
$ pnpm lint
```

## build.

```bash
$ pnpm build
```

Additional `postbuild` script runs _automagically_ and uses `next-sitemap` to generate `sitemap.xml` and `robots.txt` files.

```bash
$ pnpm postbuild
```
